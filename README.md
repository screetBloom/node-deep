- `检查当前源`: [registry-url](https://www.npmjs.com/package/registry-url)
- `判断当前是否是有网络状态`: [is-online](https://www.npmjs.com/package/is-online)
- `执行shell脚本`: [execa](https://www.npmjs.com/package/execa)
